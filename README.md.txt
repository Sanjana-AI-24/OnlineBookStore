# Online Book Store 📚

A full-stack web application built using **Three-Tier Architecture** for managing and purchasing books online.

## 🚀 Features
- **User Authentication:** Secure login and registration.
- **Dynamic Catalog:** Browse books fetched dynamically from MySQL.
- **Shopping Cart:** Add, remove, and manage book selections.
- **Order Processing:** Seamless checkout flow.

## 🛠 Tech Stack
- **Backend:** Java Servlets, Java EE
- **ORM:** Hibernate (Object-Relational Mapping)
- **Frontend:** JSP, HTML Frames, CSS
- **Database:** MySQL
- **Server:** Apache Tomcat

## 🔧 Setup Instructions
1. Clone the repository: `git clone https://github.com/your-username/OnlineBookStore.git`
2. Import the database schema using the `database_setup.sql` file.
3. Configure your MySQL credentials in `src/main/resources/hibernate.cfg.xml`.
4. Deploy the project on **Apache Tomcat**.
